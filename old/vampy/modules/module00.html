Preparation

Complete these preparation tasks before starting the AI Beginner Python course

Overview

Welcome to the AI Python 101 crash course! This guide outlines the steps you that are encoursged to complete before starting the course to get the most out of it and have smooth learning experience.

By the end of this course, :

Python Programming

you'll be able to write Python code, utilizing syntax, data structures, and basic algorithms to solve problems.

Google Colab

you'll be adept at using Google Colab to execute Python code and collaborate effectively on projects with peers.

Cursor AI

you'll be able to leverage AI-assisted coding tools like Cursor AI to boost your productivity and enhance your learning experience.

Discord

you'll be familiar with engaging with a community for support, collaboration, and knowledge sharing through platforms like Discord.

OpenAI & HuggingFace

you'll be able to explore and utilize advanced AI models for tasks such as text generation and data analysis.

UI Development

you'll be able to create intuitive and user-friendly interfaces for AI applications.

Required Steps

Step 1: Read the Course Manual

  • Go through the course manual (5 pages)
  • Take notes on key concepts, terms, and tools
  • Time estimate: 20-30 minutes

Step 2: Set Up Google Colab

  • Open this Google Colab notebook and run the code
  • Time estimate: 15-20 minutes

Step 3: Install Cursor AI

  • Download and install Cursor AI from this link (see below)
  • Create a file as described here and run a test "Hello World" script
  • Watch the setup video tutorial
  • Time estimate: 30-40 minutes

Step 4: Join Discord Community

  • Join our Discord server where all course communication happens
  • read the guidelines in the #rules channel
  • Introduce yourself in the #introductions channel
  • Time estimate: 10-15 minutes
